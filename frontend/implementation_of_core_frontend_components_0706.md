## 프론트엔드 핵심 컴포넌트 구현 및 보완

* **`NewsPage.jsx` 보완:**
  * 네이버 뉴스 API 특성상 요약문이 잘릴 수 있는 리스크를 방지하기 위해 *"※ 뉴스 요약이 잘려 보일 수 있습니다. 정확한 내용은 원문을 확인해주세요."* 경고 안내 문구 추가.
  * 원문 링크 데이터가 없는 예외 케이스 발생 시, 버튼을 숨기는 대신 사용자가 직관적으로 인지할 수 있도록 `disabled` 및 `cursor: not-allowed` 속성을 활용한 '원문 없음' 회색 버튼 활성화 UI 적용.
* **`VideoPage.jsx` 보완:**
  * 역할 지침서에 맞춰 영상 제목 외에 '설명' 및 '출처(파란색 강조 스타일)' 레이아웃 추가 배치.
  * 외부 유튜브 영상 로드 실패 시 화면이 완전히 깨지거나 빈 공간으로 남는 현상을 방지하기 위해 iframe 하단 레이어에 "현재 영상을 불러올 수 없습니다."라는 대체 안내 문구(Fallback UI)를 CSS relative 트릭으로 구현.
  * 주의사항인 '자동 재생 금지'를 완벽하게 준수하기 위해 URL 파라미터에서 자동 재생 속성을 원천 배제.
