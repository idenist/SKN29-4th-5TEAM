## 뉴스 API 백엔드 연동을 위한 기획 및 자원 확보

* **3차 프로젝트 자산 연계 뉴스 키워드 도출:** 네이버 전체 뉴스를 무작정 가져올 경우 발생하는 소모적 할당량 낭비 및 정확도 저하를 방지하기 위해, 3차 프로젝트 데이터(26,803건)의 카테고리와 매칭되는 **8개 정예 키워드 기획** 확정.
  * *키워드:* 청년 월세, 청년도약계좌, 청년 일자리, 국민내일배움카드, 청년 창업지원, K-Startup, 청년 정책, 청년 복지
* **네이버 오픈 API 신청 및 환경 세팅:** 네이버 개발자 센터에서 비로그인 오픈 API 서비스 환경을 `WEB 설정`으로 지정하고, 로컬 가상 주소인 `http://localhost:5173`을 보안 검문소 주소로 선행 등록하여 **`Client ID` 및 `Client Secret` 발급 성공**.
* **백엔드 전처리 및 응답 계약(Contract) 공유:** 발급된 키 공유와 동시에 백엔드 팀이 처리해야 할 3대 전처리 기준(필수 필드 추출, 정규표현식을 통한 HTML 태그 제거 및 텍스트 정규화, 제목 기준 중복 기사 제거) 및 프론트가 찌를 최종 엔드포인트 주소(`/api/news/`)를 정의하여 팀 단톡방에 업무 공유 완료.
